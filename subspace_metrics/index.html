This Rust module serves as a bridge between two different Prometheus metrics libraries used within our frameworks — Substrate and Libp2p. The module exposes a web server endpoint at “/metrics” that outputs metrics in Prometheus format. It adapts metrics from either or both of the following libraries:

  • Official Rust Prometheus client (registry aliased as Libp2pMetricsRegistry)
  • TiKV’s Prometheus client (registry aliased as SubstrateMetricsRegistry)
